PENTRU ROUTER DE PE VLANURI+NORMALE
1.adaugare rutare pentru vlanuri
interface g0/1.10(numarul vlanului)
description Legatura cu VLAN10
encapsulation dot1q 10(numarul vlanului)
ip address 132.172.112.1 255.255.248.0 -> adresa router ului pe vlan
exit
interface g0/1.20
description Legatura cu VLAN20
encapsulation dot1q 20
ip address 132.172.64.1 255.255.224.0
exit
interface g0/1.30
description Legatura cu VLAN30
encapsulation dot1q 30
ip address 132.172.120.1 255.255.252.0
exit
interface g0/1
no shutdown

2.adaugare comenzi dhcp
routere ce nu sunt direct la server:
pe interfata conectata catre lan/vlan!
ip helper-address DEFAULTGATEWAY address(Server)

router conectat la lan server:
(in terminal)
ip dhcp excluded-address 129.128.64.1 129.128.65.160(defaultgateway ipul, ip-ul primului host-1)
ip dhcp pool STANLEY1(nume lan)
network 129.128.64.0 255.255.224.0 (NA lan)
default-router 129.128.64.1 (defaultgateway, adresa router lan/vlan)
dns-server 129.128.111.254 (adresa server)
domain-name example.com

3.rutare cu celelalte lanuri eigrp/rip/ospf
	->EIGRP
router eigrp 1
no auto-summary
network NA IP + wildcard_mask(se trec toate ipurile ce intra in router si wildcardul este 255.255.255.255-masca pe care o avem)
passive -interface X (X=toate porturile NEFOLOSITE)

	->RIP
router rip
version 2
network NA IP(se trec toate ipurile ce intra in router)
no auto-summary

	->OSPF
router ospf 1
network N.A. + wilcard_mask area 0 (NA si wildcard la fel cca EIGRP)
passive -interface X(X=toate porturile NEFOLOSITE)
